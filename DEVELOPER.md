## Project Structure
- `/images` - Planet images
- `/audio` - Audio descriptions
- `data.json` - Content structure

## How to Add New Content
1. Edit `data.json` following existing format:
```json
{
  "title": "Planet Name",

  "audio": "path/to/audio.mp3"
}
